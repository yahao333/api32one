## 一、信息收集

api32one 承诺保护用户隐私。本服务收集的信息仅用于提供服务质量和安全防护：

- **API 使用记录**：包括请求时间、模型类型、token 消耗量，用于用量统计和账户管理
- **技术日志**：包括请求参数、响应状态、错误信息，用于问题排查和服务优化
- **激活信息**：包括兑换码、使用额度、账户状态，用于用户身份验证和服务管理

> **我们不收集**：您的业务数据内容、API 密钥明文（仅存储哈希值用于验证）、或任何与您业务相关的敏感信息。

## 二、数据处理

您的请求数据处理流程如下：

1. 您的应用将请求发送至 api32one
2. api32one 进行敏感词检测和个人信息（PII）脱敏处理
3. 处理后的请求被转发至相应的第三方大模型提供商
4. 第三方返回的响应通过 api32one 中转回您的应用

## 三、第三方数据共享

为完成 API 请求，您的数据将被发送至您选择的第三方大模型提供商。我们会与以下类型的第三方共享数据：

- OpenAI (ChatGPT / GPT-4 系列)
- Anthropic (Claude 系列)
- Google (Gemini 系列)
- DeepSeek (DeepSeek Chat 系列)

请参阅各第三方提供商的隐私政策了解其数据处理实践。

## 四、数据安全

我们采用以下措施保护您的数据安全：

- HTTPS 加密传输，防止中间人攻击
- 敏感词自动检测，拦截违规内容
- 个人信息（PII）自动脱敏，保护用户隐私
- API 密钥安全存储，不明文暴露
- 完整的操作日志，支持安全审计

## 五、数据保留

我们保留 API 使用记录的最长期限为 12 个月，用于用量统计和争议解决。超过保留期限的数据将被安全删除。

账户相关数据在账户注销后保留 90 天，随后删除。

## 六、您的权利

作为用户，您享有以下权利：

- **知情权**：了解您的数据如何被收集和处理
- **访问权**：获取您的 API 使用记录
- **删除权**：要求删除您的账户及关联数据
- **投诉权**：对数据处理行为向相关监管机构投诉

## 七、Cookie 使用

本服务不使用 Cookie 进行用户追踪。

## 八、儿童隐私

本服务不面向 18 岁以下的未成年人设计，也无意收集任何未成年人的个人信息。

## 九、政策更新

我们可能会不时更新本隐私政策。重大变更将在 GitHub 仓库中公告。如您继续使用服务，即表示您接受更新后的政策。

## 十、联系我们

如您对本隐私政策有任何疑问或顾虑，请通过 [GitHub Issues](https://github.com/yahao333/api32one/issues) 与我们联系。
